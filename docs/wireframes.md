## 1. 공통 레이아웃

### 1.1 데스크탑 레이아웃

```
┌──────────────────────────────────────────────────────────────┐
│  [📚 woojuLib]              [🔍 검색]      [🔔] [🌸 샛별 ▾]   │  ← Header
├────────────┬─────────────────────────────────────────────────┤
│            │                                                  │
│  📊 대시보드 │                                                  │
│  📖 내 서재  │                                                  │
│  ✍️ 기록하기 │              [ 메인 콘텐츠 영역 ]               │
│  💬 리뷰     │                                                  │
│  ✨ 추천     │                                                  │
│  👨‍👩‍👧‍👦 가족   │                                                  │
│  ⚙️ 설정     │                                                  │
│            │                                                  │
│  Sidebar   │                                                  │
└────────────┴─────────────────────────────────────────────────┘
```

### 1.2 모바일 레이아웃

```
┌──────────────────────────┐
│ [📚]    [🔍]  [🔔] [🌸] │  ← Header
├──────────────────────────┤
│                          │
│                          │
│    [ 메인 콘텐츠 ]       │
│                          │
│                          │
├──────────────────────────┤
│ [📊] [📖] [✍️] [💬] [👨‍👩‍👧‍👦]│  ← Bottom Navigation
└──────────────────────────┘
  대시보드 서재 기록 리뷰 가족
```

**Header 구성 요소:**
- 로고 (woojuLib) - 클릭 시 대시보드로 이동
- 검색 바 - 도서 검색
- 알림 아이콘 - 미확인 알림 뱃지 표시
- 프로필 드롭다운 - 아바타 + 닉네임, 설정/로그아웃

### 2.2 대시보드 영역별 상세 정보

#### A. 인사 헤더 (Greeting Header)
| 정보 | 내용 |
|------|------|
| 환영 메시지 | "안녕하세요, {닉네임}님! {이모지}" |
| 가족 요약 | "이번 달 가족이 함께 {N}권을 읽었어요" |
| 현재 날짜 | 년/월 표시 |

#### B. 요약 통계 카드 (Summary Stat Cards)
| 카드 | 표시 정보 |
|------|----------|
| 읽은 책 | 이번 달 완독 권수 / 목표 권수 |
| 독서 시간 | 누적 독서 시간 (분/시간) |
| 독서 목표 | 진행률 바 + 퍼센트 + 격려 메시지 |

#### C. 월별 독서량 추이 차트 (Monthly Trend Chart)
- **타입**: 막대 그래프 (Bar Chart)
- **X축**: 최근 5~6개월
- **Y축**: 읽은 권수
- **인터랙션**: 막대 호버 시 정확한 수치 툴팁
- **필터**: 나만 보기 / 가족 전체 토글

#### D. 장르별 분포 차트 (Genre Distribution)
- **타입**: 파이 차트 / 도넛 차트
- **데이터**: 읽은 책의 장르 비율
- **범례**: 장르명 + 퍼센트
- **컬러**: 장르별 고유 색상

#### E. 가족 랭킹 (Family Ranking)
| 정보 | 내용 |
|------|------|
| 순위 | 🥇🥈🥉 메달 아이콘 |
| 구성원 | 아바타 + 닉네임 (구성원 컬러) |
| 독서량 | 이번 달 읽은 권수 |
| 강조 | 1위 하이라이트 |

> 💡 경쟁이 아닌 '함께'의 의미로 격려 메시지 포함

#### F. 가족 활동 피드 (Family Activity Feed)
각 활동 아이템에 포함되는 정보:
| 정보 | 내용 |
|------|------|
| 구성원 | 아바타 + 닉네임 (컬러) |
| 활동 타입 | 완독 / 리뷰 작성 / 읽기 시작 / 좋아요 |
| 도서 정보 | 책 제목 (클릭 시 상세) |
| 부가 정보 | 별점, 리뷰 미리보기 |
| 시간 | 상대 시간 (2시간 전, 어제) |
| 상호작용 | 좋아요(♥) 수, 댓글(💬) 수 + 버튼 |

**활동 타입별 아이콘:**
- ✅ 완독: "○○님이 「책」을 완독했어요!"
- ✍️ 리뷰: "○○님이 리뷰를 작성했어요"
- 📖 시작: "○○님이 「책」을 읽기 시작했어요"
- ✨ 추천: "○○님이 「책」을 추천했어요"

#### G. 오늘의 추천 도서 (Recommendation Carousel)
| 정보 | 내용 |
|------|------|
| 표지 이미지 | 도서 커버 |
| 제목 | 책 제목 |
| 추천 이유 | "우주님이 좋아할 만한 책" |
| 인터랙션 | 좌우 스크롤, 클릭 시 상세 |

> 도서를 검색하고 독서 상태/기록을 작성하는 페이지

```
┌──────────────────────────────────────────────────────────────┐

│                                                              │
│  어떤 책을 읽으셨나요?                                        │
│                                                              │

│  │ 🔍 책 제목, 저자, ISBN으로 검색...        │  │ 검색   │  │  ← 검색 바
│  └──────────────────────────────────────────┘  └────────┘  │
│                                                              │
│  ─────────── 검색 결과 ───────────                          │
│                                                              │
│  ┌────────────────────────────────────────┐                │
│  │ [표지] 해리포터와 마법사의 돌           │                │  ← 검색 결과
│  │ [이미지] J.K. 롤링 · 문학수첩           │   [선택]      │     리스트

### 3.3 독서 기록 영역별 상세 정보

#### A. 도서 검색 영역
| 정보 | 내용 |
|------|------|
| 검색 입력 | 제목/저자/ISBN 검색 |
| 검색 소스 | Google Books, Naver, Kakao API |
| 결과 표시 | 표지, 제목, 저자, 출판사, 출판년도, 페이지 수 |
| 수동 입력 | API에 없는 책 직접 등록 옵션 |

#### B. 선택된 도서 정보
| 정보 | 내용 |
|------|------|
| 표지 이미지 | 도서 커버 |
| 기본 정보 | 제목, 저자, 출판사, 페이지 수 |
| 장르 태그 | 자동 분류된 장르 |
| 변경 버튼 | 다른 책으로 다시 선택 |

#### C. 독서 상태 선택 (필수)
| 상태 | 설명 | 후속 필드 |
|------|------|----------|
| 읽고 싶어요 | 위시리스트 | 날짜/별점 숨김 |
| 읽는 중 | 진행 중 | 시작일, 현재 페이지 |
| 완독 | 다 읽음 | 시작/완료일, 별점, 리뷰 |

> 💡 상태에 따라 입력 필드가 동적으로 표시/숨김

#### D. 독서 정보 입력
| 필드 | 필수 여부 | 설명 |
|------|----------|------|
| 읽은 기간 | 선택 | 시작일/완료일 날짜 선택 |
| 읽은 페이지 | 선택 | 슬라이더 또는 숫자 입력, 진행률 표시 |
| 별점 | 선택 | 1~5점 별 클릭 |
| 독서 시간 | 선택 | 분 단위 입력 |

#### E. 감상평 작성 (선택)
| 필드 | 설명 |
|------|------|
| 감상평 | 자유 텍스트 (placeholder로 질문 유도) |
| 스포일러 체크 | 스포일러 포함 시 가족에게 경고 표시 |

#### F. 저장 액션
- 저장 시 → 가족 활동 피드에 자동 등록
- 완독 시 → 축하 애니메이션 (🎉 컨페티)
- 목표 달성 시 → 달성 알림

### 3.4 모바일 독서 기록 (단계별 진행)

```

│ ┌──────────────────────┐ │      │ │표지│ J.K.롤링        │

│ └──────────────────────┘ │      ├──────────────────────────┤
│ ┌──────────────────────┐ │      │ 📌 상태                  │
│ │[표지] 해리포터       │ │ ───→ │ [싶어][읽는][완독✓]     │
│ │ J.K.롤링      [선택] │ │      │ ⭐ ☆☆☆☆☆            │

## 5. 상태 흐름 (State Flow)

### 5.1 독서 기록 작성 플로우

```
[검색] → [도서 선택] → [상태 선택] → [정보 입력] → [저장]
                                                      │
                                          ┌───────────┴───────────┐
                                          ▼                       ▼
                                   [활동 피드 등록]        [완독 시 축하 🎉]
                                          │                       │
                                          ▼                       ▼
                                   [가족에게 알림]         [목표 달성 체크]
```

### 5.2 대시보드 데이터 로드

```
[페이지 진입]
      │
      ├── 사용자 통계 조회 (React Query)
      ├── 가족 통계 조회
      ├── 활동 피드 조회 (페이징)
      └── 추천 도서 조회
      │
      ▼
[로딩 스켈레톤 표시] → [데이터 렌더링]
```
